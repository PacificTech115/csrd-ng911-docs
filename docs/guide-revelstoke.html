  Workflow Walkthrough (Revelstoke)

  Purpose and Scope

This guide explains the standard Revelstoke workflow for editing the NG911 central addressing data:

  1. Log in to the CSRD portal.
  2. Open the Revelstoke NG911 group.
  3. Open the NG911_Address_Revelstoke_Edit feature layer.
  4. Choose Open in ArcGIS Pro.
  5. Edit attributes/geometry and save edits.
  6. Verify changes in the Feature Layers Data tab on the CSRD Portal.

  Step-by-Step Editing Workflow

Step 1: Log in to the CSRD portal

  1. Open a web browser and navigate to the CSRD Portal Home URL.
  2. Click Sign In in the top-right corner.
  3. Enter your assigned username and password.
  4. Confirm you are logged in successfully before proceeding.

Step 2: Navigate to the Revelstoke NG911 group

  1. From the Groups tab, locate the Revelstoke NG911 group.
  2. Verify the page shows the Revelstoke group content list.

Step 3: Open the editing feature layer item

  1. In the group content, click the NG911_Address_Revelstoke_Edit item.
  2. Review the item details to confirm it is the correct editing layer.
  3. Confirm layer ownership and metadata match Revelstoke editing workflow.

Step 4: Open in ArcGIS Pro

  1. On the item page, click the highlighted Open in ArcGIS Pro action on the right.
  2. Allow ArcGIS Pro to launch (or switch to it if already open).
  3. Sign in to ArcGIS Pro if prompted to authenticate against the CSRD portal.
  4. Confirm the layer loads successfully into your map/project.

Step 5: Edit records

  1. In ArcGIS Pro, open the Edit tab and start an edit session if required.
  2. Use the Modify Features pane to select an existing point to update fields, or create a new address point if needed.
  3. Update required addressing fields according to local standards.
  4. Validate geometry placement and attribute completeness before saving.

Step 6: Save edits

  1. Click Save in the ArcGIS Pro Edit tab to commit changes.
  2. Confirm no validation errors remain from attribute rules.
  3. Optionally review the updated feature in the attribute table or map popup.

Step 7: Verification of changes

  1. Return to the CSRD Portal in your web browser.
  2. Navigate to the Revelstoke editing feature layer item page.
  3. Open the Data tab.
  4. Review the records to confirm your recent edits from ArcGIS Pro are successfully reflected in the central portal.

Quality Checklist Before Save

  • Correct layer: NG911_Address_Revelstoke_Edit
  • Correct account: Editor account for edits
  • Required fields populated: Agency, Address Number, Street Name, Locality
  • No obvious typos in civic address or locality
  • Point location is accurate on the building footprint

Troubleshooting

  • Confirm username/password are entered exactly (case-sensitive).
  • Ensure you are using the correct role account (Viewer vs Editor).
  • Retry sign-in from the CSRD portal home page.
  • Ensure ArcGIS Pro is installed on the workstation.
  • Ensure the machine is associated with the correct portal/account access.
  • Try launching ArcGIS Pro first, then re-click Open in ArcGIS Pro.
  • Verify you are signed in with the Editor account.
  • Check required fields and geometry validity.
  • Confirm the layer is editable and not opened read-only.
