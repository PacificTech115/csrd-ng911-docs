ArcGIS GP Tools

3 server-side geoprocessing services for database operations
These GP services run server-side on ArcGIS Server and are called by the Nightly Orchestrator notebook. They operate directly on the enterprise geodatabase.

QA Validation Tool

GP Service

Schema validation, NGUID checks, mandatory nulls, duplicate detection

Reconcile/Post Tool

GP Service

Traditional versioning reconcile/post with 3 stage modes

Export Tool

GP Service

Exports DEFAULT to FGDB ZIP, copies to network share

How GP Tools Fit the Pipeline

The Nightly Orchestrator notebook (an ArcGIS Notebook Server script) calls these GP tools as server-side operations via their REST endpoints. Each tool runs within the ArcGIS Server context with direct SDE access to the enterprise geodatabase.
